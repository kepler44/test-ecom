---

## What i used?

```
react-17, router-6, redux-saga, bootstrap-5, axios, express, nodejs-17, firebase.
```

---

## Run

You can run it with the **dev** command.

```
yarn install
```

```
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
